## ✔️ 커밋 규칙
- `week1`, `week2`, `week3` 폴더를 생성하고 그 안에 코드 업로드

- 커밋명은 풀었으면 `Solved`, 못 풀었으면 `Solving`
- 푼 문제 주소는 본인 폴더의 `README.md` 표에 작성  
  - 예시:
    ```
    [A+B](https://www.acmicpc.net/problem/1000)
    ```
